Команди Git

Base

git config --global user.name blabla - встановлює ім'я, яке буде відображатисяgit config --global user.email blabal@bla - встановлює пошту,яка буде відображатися git init - створює новий локальний репозиторій в поточній директорії git status -перераховує всі нові або змінені файли, які потребують в фіксації git add - додає файли в коміт. Зазвичай використовується "." або "*" для додавання всіх файлів. Щоб додати один файл потрібно вказати його ім'я та розширення. git commit - робить коміт проекту git commit -m - аргумент коміта з допомогою якого можна зробити опис( що було змінено або додано в цьому коміті)git log -показує історію комітів(те що робили в Git) git checkout -переключається на вибрану гілку і оновлює робочу папку до її стану git branch - виводить всі доступні коміти git push - пушить(відправляє) коміт з локального репозиторію у віддалений git pull - завантажує дані з віддаленого репозиторію у локальний git merge - об'єднує дві гілки в одну git clone - завантажує репозиторій та його файли зі всією історією змін .gitignore - файл, який приховує певні файли від системи Git
